# Privacidad

Mapa Comunitario esta pensado como una herramienta local y liviana.

La aplicacion sirve para encontrar vias de contacto. No solicita ni necesita nombres, cedulas, historias clinicas u otros datos personales para buscar recursos.

- No envia notas, favoritos ni recursos agregados a un servidor propio.
- No requiere crear cuenta.
- No usa base de datos remota.
- Guarda datos locales en `localStorage`, dentro del navegador.
- Cuando se usa "Buscar online", la zona escrita se envia a servicios publicos de OpenStreetMap para ubicar recursos cercanos.
- Cuando se informa un dato desactualizado, GitHub abre un formulario externo y aplica su propia politica de privacidad.

Si se publica una version institucional o multiusuario, revisar esta politica antes de recolectar datos personales.

Evitar guardar en notas locales informacion sensible de personas usuarias, historias clinicas, datos judiciales o identificadores personales innecesarios.

Los reportes publicos de datos desactualizados tampoco deben contener informacion personal o detalles de una situacion individual.
